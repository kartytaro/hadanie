# The Rider–Waite Historical Codex

A lightweight, single-file interactive Tarot codex and reading companion. Built as a standalone HTML application, it is designed for ease of use, serving as both reference for historical card interpretations and a functional tool for personal readings or solo tabletop RPG sessions.

## Features

- **Historical Interpretations:** Features a curated collection of card meanings drawn from foundational cartomantic and esoteric texts, including Pratesi's Cartomancer (1750), De Mellet (1781), Éliphas Lévi, Paul Christian, MacGregor Mathers, the Hermetic Order of the Golden Dawn, and A.E. Waite.
- **Dual Views:** - *Codex View:* Browse and study the complete Major and Minor Arcana.
  - *Reading View:* Draw cards for practical divination or randomized narrative prompts.
- **Bilingual Interface:** Built-in localization toggling (EN/RU).

## Use Cases

Beyond standard cartomancy, this codex is highly optimized to be kept open in a side-monitor or tablet as a quick-reference tool for:
- Studying the evolution of Tarot interpretations across different centuries.
- Serving as an automated oracle or prompt-generator for solo journaling games and tabletop RPGs.
- Quick, programmatic card generation without shuffling a physical deck.
